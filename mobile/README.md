# Cresca Movement DeFi - React Native Mobile App

This is the React Native mobile application for the Movement DeFi platform. It will include all 12 screens and features from the web version, rebuilt for iOS and Android using React Native and TypeScript.

## Planned Features
- Landing (Onboarding)
- Dashboard
- Market
- Wallet
- Settings
- History
- Basket Builder
- Review Modal
- Position Details
- Close Position Modal
- Success Modal
- Deposit
- Withdraw
- Swap

## Stack
- React Native (Expo or CLI)
- TypeScript
- React Navigation
- UI Library: React Native Paper or NativeBase

## Setup (to be updated)
```sh
# Install dependencies
npm install
# Start development server
npm start
```

---

This app will be developed in the `/mobile` directory alongside the web app.
